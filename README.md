🧠 Design of a CNN–Swin Transformer Model for Alzheimer's Disease Prediction

A Final Year Project that leverages Deep Learning and Computer Vision techniques to assist in the early detection of Alzheimer's Disease from MRI brain images. The project combines the feature extraction capabilities of Convolutional Neural Networks (CNN) with the contextual learning power of the Swin Transformer to improve disease classification performance.

📖 Project Overview
Alzheimer's Disease is a progressive neurological disorder that affects memory and cognitive function. Early diagnosis is essential for timely treatment and better patient care.
This project presents a hybrid deep learning model that combines CNN and the Swin Transformer to classify MRI brain images into different Alzheimer's disease stages. The application also includes a Flask-based web interface for user interaction.

🎯 Objectives
Detect Alzheimer's Disease from MRI brain images.
Improve prediction accuracy using a hybrid CNN–Swin Transformer architecture.
Provide a user-friendly web interface for prediction.
Assist healthcare professionals with AI-based disease classification.

✨ Features
MRI image preprocessing
Hybrid CNN–Swin Transformer model
Flask-based web application
Disease stage prediction
Database integration
Simple and interactive user interface

⚙️ Installation
Clone the repository.
Navigate to the Code folder.
Install the required packages:
pip install -r requirements.txt
Run the Flask application:
python app.py
Open the application in your browser.

📊 Results
The hybrid CNN–Swin Transformer model demonstrates promising performance in classifying Alzheimer's Disease from MRI brain images. The project aims to improve diagnostic support by combining deep learning techniques with a web-based interface.
